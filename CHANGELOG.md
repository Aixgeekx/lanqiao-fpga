# 更新日志

本项目按语义化版本管理：`主版本.次版本.修订版本`。

## [0.3.0] - 2026-05-30

- PDF 视觉全面优化：
  - 代码块改为正体 MapleMono 字体 + 深色背景容器（#2d3748），告别白底代码
  - 章节标题添加左侧彩色竖条装饰（ChapterSideBar），提升视觉层次
  - 封面定位更新为"国一工程模板 · 零基础教学 · 赛场速查手册"
- PDF 功能增强：
  - 添加 7 个 PDF 书签（前言 + 5 章 + 附录），支持 PDF 阅读器快速导航
  - 添加 PDF 元数据（标题、作者、主题、关键词）
- 代码质量优化：
  - 提取 `std_table_style()` 辅助函数，消除 23 处重复表格样式定义
  - 提取 `captioned_table()` 辅助函数，支持表格+标题 KeepTogether 防断页
  - 提取 `code_block()` 辅助函数，统一代码块深色背景渲染
  - 提取 `chapter_title_block()` 辅助函数，统一章节标题样式
- 扫描图片质量提升：JPEG 压缩质量从 78 提升到 85，中文小字更清晰
- 当前 PDF：137 页，6.7 MB，正文可抽取字符 141110 个

## [0.2.1] - 2026-05-29
