# 更新日志

本项目按语义化版本管理：`主版本.次版本.修订版本`。

## [0.2.4] - 2026-05-30

- 教材 PDF 增强到 121 页，正文可抽取字符 140899 个，表格编号 39 个，题面图片编号 46 个。
- 新增“上板故障定位流程”，把 bit 生成失败、下载失败、基础IO异常、外设无响应、业务状态不走和偶发死机分层排查。
- 新增“调试信号分配建议”，给 LD1-LD8 分配心跳、按键脉冲、状态位、busy、error 和 done 等赛场可观察信号。
- 新增“真题纸面批注模板”和“真题批注完成检查”，把扫描题面转化为端口、时间参数、状态机、协议帧、显示格式和提交风险记录。
- 重新生成并渲染检查 PDF，更新完整发行包 `lanqiao-fpga-textbook-v0.2.4.zip` 与轻量发行包 `lanqiao-fpga-textbook-pdf-v0.2.4.zip`。
